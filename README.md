# Taxi-Analysis

本项目基于微软 T-Drive 数据集（10,375 辆出租车 GPS 轨迹），实现一个具备海量数据处理、时空检索及热力图可视化功能的分析系统。

## 时间安排
- [ ] **Phase 1 (3.12 - 3.22):** 基础架构搭建。实现单车轨迹解析与 Qt 基础窗口渲染。
- [ ] **Phase 2 (3.23 - 4.14):** 数据性能优化。建立全局索引，实现全城网格化热力图 (F4)。
- [ ] **Phase 3 (4.15 - 5.06):** 核心算法攻坚。实现区域检索 (F3) 与频繁路径挖掘 (F7/F8)。
- [ ] **Phase 4 (5.07 - 5.24):** 系统集成与交付。性能压力测试，录制演示视频，撰写结项报告。
